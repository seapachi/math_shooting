## プロジェクト概要
「Math Lane Shooter」は、4レーンに落下してくる答えの中から正解を選ぶ、計算×シューティングのミニゲームです。  
プレイヤーはレーン移動と発射で正解を狙い、スコアを伸ばします。

## 遊び方（基本）
- 画面上部に表示される計算式の答えが、4つのレーンに落下します。
- タップでレーン移動、左右スワイプでレーン移動ができます。
- 素早いダブルタップで発射し、狙ったレーンの答えを撃ち抜きます。
- 正解でスコア+1、間違いや見逃しでライフが減ります。

## 画面構成
- **Start画面**: タイトル、遊び方、STARTボタン。
- **Game画面**: 問題表示、スコア/ライフ、レーン、落下する選択肢、発射処理。
- **Pause画面**: 一時停止と再開/終了。
- **Game Over画面**: 最終スコア表示とリトライ。

## 実装の要点
- すべて `index.html` に Phaser 3 を使って実装。
- 視覚デザインは `styles.css` に集約。
- 最近の変更で「宇宙×シューティング」イメージへ寄せるため、
  フォント・配色・背景グラデーション・星空演出を強化。

## ファイル構成
- `index.html`: ゲーム本体（Phaser 3 のシーン/ロジック/描画）
- `styles.css`: デザイン（テーマ色・背景・装飾）
